# Custom baseline CNN dir structure

## Project Structure

```
deep-learning/
│
├── data/                     # Dataset directory
│   ├── train/                # Training data
│   ├── val/                  # Validation data
│   └── test/                 # Test data
│
├── Custombaselinecnn/                     # Custom CNN model implementation
│   ├── baselinecnntrain.ipynb             # Model training notebook
│   ├── baseline_eval.py                   # to run the evaluation 
│   ├── hyperpara_batch_tuning.ipynb       # batch hyperparameter tuning notebook
│   ├── hyperpara_lr_tuning.ipynb          # learning rate hyperparameter tuning notebook           
│   ├── bestbaselinecnn.pth                # batch hyperparameter tuning notebook
│   └── requirement.txt                    # all dependenices required     
│

```
